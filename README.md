個人間で匿名で気軽に質問、回答ができます。


 # **URL**
[Heroku](https://usamimi.herokuapp.com/)


# **使用技術**
- Ruby 3.1.4
- Ruby on Rails 6.1.7
- Puma
- Bootstrap
- RSpec
- Rubocop
- SQLite3
- PostgreSQL
- AWS S3
- Heroku
- Docker/Docker-compose


# **機能一般**
- ユーザー登録、ログイン機能(Devise)
- 質問機能
- 回答機能
- ベスト回答決定
- ソート機能
- コメント機能
- 検索機能


# **テスト**
